### **SECTION 1 — CORE: Condensed Syllabus (7-Row Table)**

|**Topic**|**Subtopics**|
|---|---|
|**1.1 Framework Philosophy & Setup**|Framework philosophy & use cases; mobile-first design; Bootstrap vs custom CSS; installation via CDN, npm/yarn, Sass; compiled vs source assets; file structure & architecture; customization entry points; browser support; polyfills & fallbacks; cross-browser testing; graceful degradation; JavaScript dependencies; Popper.js; bundle vs separate JS; module vs UMD builds.|
|**1.2 Grid System & Responsive Layouts**|Containers (`.container`, `.container-fluid`, responsive variants); max-widths; padding; breakpoints xs–xxl; mobile-first tiers; 12-column grid; auto & fixed columns; responsive spans; gutters (`.g-*`, `.gx-*`, `.gy-*`, `.g-0`); ordering (`.order-*`); offsets (`.offset-*`); row columns (`.row-cols-*`); nesting grids; vertical alignment (`.align-items-*`, `.align-self-*`); horizontal distribution (`.justify-content-*`); centering; flex integration.|
|**1.3 Typography & Content Styling**|Headings & display text; typographic hierarchy; paragraphs & `.lead`; font sizing & line height; inline text utilities (bold, italic, underline, strike, mark, small); semantic vs presentational styling; text alignment & responsive alignment; text transformation; lists (unstyled, inline, definition); list groups; blockquotes & citations; footer attribution.|
|**1.4 Color System & Utilities**|Theme & semantic colors (primary–dark); contextual usage; background utilities (`.bg-*`, gradients, opacity); accessible text-background pairs (`.text-bg-*`); text colors & opacity; emphasis & muted text; link colors; border colors & sides; border width/style; WCAG contrast ratios; accessibility testing & helpers.|
|**1.5 Spacing & Sizing Utilities**|Margin & padding utilities (`.m-*`, `.p-*`); directional spacing; scale 0–5; negative margins; auto centering; responsive spacing; width & height utilities (percent, viewport, max-width, auto); gap utilities (`.gap-*`, row/column gaps); gap vs gutter distinction.|
|**1.6 Components — Structure & Behavior**|Navigation (Navbar, Nav/Tabs, Breadcrumb, Pagination); content containers (Cards, Accordion, List Group, Modal, Offcanvas); forms (Input Groups, Controls, Checks & Radios, Range, Floating Labels, Validation); feedback (Alerts, Toasts, Progress, Spinners); interactive (Buttons, Button Groups, Dropdowns, Collapse, Carousel, Tooltip, Popover, Scrollspy); visual (Badges, Placeholders, Close button); variants, responsiveness, accessibility, JS behavior.|
|**1.7 Utility Classes & Helpers**|Display utilities; flexbox utilities; positioning (static to sticky, offsets, z-index); border utilities; shadow utilities; visibility & opacity; overflow control; floats & clearfix; interaction helpers (user-select, pointer-events, cursor); vertical alignment utilities; responsive variants across breakpoints.|
