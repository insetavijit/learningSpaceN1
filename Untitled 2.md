### **Section 1 — Core Fundamentals (7 major areas)**

- **1.1 Language Fundamentals**: Variables, data types, operators, control flow, loops, type coercion
- **1.2 Functions & Scope**: Declarations, arrows, closures, `this`, higher-order functions
- **1.3 Objects & Prototypes**: Object creation, prototypal inheritance, ES6 classes, composition
- **1.4 Arrays & Iteration**: Array methods (map, filter, reduce), modern methods, destructuring
- **1.5 Asynchronous JavaScript**: Event loop, callbacks, Promises, async/await, combinators
- **1.6 ES6+ Modern Features**: Destructuring, spread/rest, template literals, modules, generators
- **1.7 Error Handling & Debugging**: Try/catch, custom errors, DevTools, debugging techniques
