### **[[Common String Functions — Expanded Table]]*

| **Function**    | **Purpose / Use-Case Summary**                                 | **Quick Example**                     |
| --------------- | -------------------------------------------------------------- | ------------------------------------- |
| `strlen()`      | Returns string length — useful for validation and size checks. | `strlen("Hello")` → `5`               |
| `strpos()`      | Finds position of substring; `false` if not found.             | `strpos("hello","e")` → `1`           |
| `substr()`      | Extracts part of a string (start, length).                     | `substr("abcdef", 1, 3)` → `bcd`      |
| `trim()`        | Removes whitespace (or characters) from both ends.             | `trim(" hi ")` → `"hi"`               |
| `explode()`     | Splits string into array by delimiter.                         | `explode(",", "a,b,c")`               |
| `implode()`     | Joins array elements into a string.                            | `implode("-", ["a","b","c"])`         |
| `str_replace()` | Replace text within a string.                                  | `str_replace("car","bus","carwash")`  |
| `sprintf()`     | Formatted string output (templates).                           | `sprintf("Hi %s, %d pts", "Avi", 90)` |
| `json_encode()` | Converts array/object to JSON.                                 | `json_encode($data)`                  |
| `json_decode()` | Converts JSON string to array/object.                          | `json_decode($json, true)`            |
### **Array Functions — Expanded Table*

|**Function**|**Purpose / Use-Case Summary**|**Quick Example**|
|---|---|---|
|`array_map()`|Apply a callback to each element — transform values.|`array_map(fn($x)=>$x*2, [1,2,3])` → `[2,4,6]`|
|`array_filter()`|Filter items based on condition — keeps matching values.|`array_filter([1,2,3], fn($x)=>$x>1)` → `[2,3]`|
|`array_reduce()`|Reduce array to a single value (sum, total, etc).|`array_reduce([1,2,3], fn($c,$i)=>$c+$i)` → `6`|
|`array_merge()`|Merge arrays together — indexing resets numeric keys.|`array_merge([1,2],[3])` → `[1,2,3]`|
|`sort()`|Sort array by values (ascending). **Reindexes** numeric keys.|`sort($arr)`|
|`ksort()`|Sort by **keys**, keeps associations.|`ksort($data)`|
|`isset()`|Checks if key exists **and** value is not null.|`isset($arr['x'])`|
|`in_array()`|Check if value exists in array.|`in_array('php', $langs)`|
|`array_key_exists()`|Checks if key exists (even if `null`).|`array_key_exists('x',$arr)`|

### **OOP Keywords — Expanded Table**

|**Keyword**|**Purpose / Use-Case Summary**|**Quick Example**|
|---|---|---|
|`class`|Defines a blueprint containing properties & methods.|`class User {}`|
|`interface`|Defines contract-only method signatures; enforces implementation.|`interface Loggable { public function log(); }`|
|`trait`|Reuse common method code across classes (horizontal inheritance).|`trait Helper { function ping(){}}`|
|`abstract`|Base class that **cannot be instantiated**; may define abstract methods.|`abstract class Shape { abstract function area(); }`|
|`final`|Prevents overriding (methods) or inheritance (classes).|`final class Config {}`|
|`extends`|Inherits from a parent class.|`class Admin extends User {}`|
|`implements`|Implements interface contracts.|`class FileLogger implements Loggable {}`|
|`namespace`|Logical grouping of classes; avoids naming conflicts.|`namespace App\Models;`|
|`use`|Import namespaces, traits, or alias names.|`use App\Services\Auth;`|
|`readonly`|Defines properties that are **write-once** (constructor-only).|`public readonly string $id;`|
